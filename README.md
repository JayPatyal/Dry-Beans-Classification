# Dry-Beans-Classification


The data selected is the “Dry Beans Dataset” which was taken from the UCI repository.
https://archive.ics.uci.edu/ml/datasets/Dry+Bean+Dataset
This dataset was built by using computer vision system developed to distinguish seven different registered varieties of dry beans.
This dataset has seven different types of beans, which has 16 different attributes (12 dimensions and 4 shape forms) of each grain. The total number of instances of this dataset are 13,611.
